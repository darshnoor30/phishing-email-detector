# Phishing Email Detector

A beginner-friendly cybersecurity project developed using Python.

## Features
- Detects phishing keywords
- Suspicious link analysis
- Risk score calculation
- GUI-based phishing detection

## Technologies Used
- Python
- Tkinter

## Purpose
This project was built to strengthen my understanding of phishing attacks, cybersecurity fundamentals, and Python GUI development.

## How to Run
1. Download or clone this repository
2. Open terminal in project folder
3. Run:
python gui.py
